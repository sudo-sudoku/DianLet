¿Qué es el Sistema 3Lite?

El Sistema Tributario 3Lite es una versión simplificada y eficiente para la gestión de casos tributarios, seguimiento de metas y registro de gestión. Diseñado específicamente para DIAN Leticia, este sistema se implementa en SharePoint y ofrece todas las funcionalidades esenciales en un paquete fácil de instalar y usar.

Implementación Rápida

Aproximadamente 45 minutos siguiendo la guía paso a paso para tener un sistema completamente funcional.

Estructura Simplificada

Solo 3 listas principales (Casos, Gestión, Metas) para mantener el sistema ligero y eficiente.

Dashboard Visual

Visualización clara de casos, metas y cumplimiento para una toma de decisiones informada.

Aplicativo de Registro

Interfaz web intuitiva para el registro rápido y eficiente de la gestión tributaria.

Alertas Automáticas

Notificaciones de casos próximos a vencer para no perder fechas importantes.
